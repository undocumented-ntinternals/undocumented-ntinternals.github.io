SEMAPHORE_BASIC_INFORMATION
SEMAPHORE_BASIC_INFORMATION

typedef struct _SEMAPHORE_BASIC_INFORMATION {



  ULONG                   CurrentCount;
  ULONG                   MaximumCount;

} SEMAPHORE_BASIC_INFORMATION, *PSEMAPHORE_BASIC_INFORMATION;



Structure SEMAPHORE_BASIC_INFORMATION is retrieved as a result of call NtQuerySemaphore with SemaphoreBasicInformation information class.


  • CurrentCount     Current state of semaphore's counter.
  • MaximumCount     Maximum counter position, defined with call to NtCreateSemaphore.
