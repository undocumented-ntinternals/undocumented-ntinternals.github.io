NtSetInformationToken(



  IN HANDLE               TokenHandle,
  IN TOKEN_INFORMATION_CLASS TokenInformationClass,
  OUT PVOID               TokenInformation,
  IN ULONG                TokenInformationLength );



Function NtSetInforamtionToken sets parameters of Token Objects. See also description of SetTokenInformation in Win32 API SDK.


  • TokenHandle     HANDLE to Token Object opened with TOKEN_ADJUST_DEFAULT access.
  • TokenInformationClass     Information class descripted in TOKEN_INFORMATION_CLASS topic.
  • TokenInformation     User's allocated buffer containing data to set to.
  • TokenInformationLength     Length of TokenInformation buffer, in bytes.
