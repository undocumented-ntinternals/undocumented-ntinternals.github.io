NtOpenKeyedEvent
NtOpenKeyedEvent

NTSYSAPI 
NTSTATUS
NTAPI


NtOpenKeyedEvent(



  OUT PHANDLE             KeyedEventHandle,
  IN ACCESS_MASK          DesiredAccess,
  IN POBJECT_ATTRIBUTES   ObjectAttributes OPTIONAL );



Function NtOpenKeyedEvent is used for open previously created KeyedEvent with associated name.
  • KeyedEventHandle     Result of call - HANDLE to opened KeyedEvent object.
  • DesiredAccess     Access to object, the same values as for Event object.
  • ObjectAttributes     Name of KeyedEvent to open.


    Supported on system versions:
    Win XP/2003
