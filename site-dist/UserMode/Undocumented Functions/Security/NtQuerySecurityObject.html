NtQuerySecurityObject(



  IN HANDLE               ObjectHandle,
  IN SECURITY_INFORMATION SecurityInformationClass,
  OUT PSECURITY_DESCRIPTOR DescriptorBuffer,
  IN ULONG                DescriptorBufferLength,
  OUT PULONG              RequiredLength );



Function NtQuerySecurityObject retrieve object's Security Descriptor.




  • ObjectHandle     HANDLE to any object opened with READ_CONTROL access.
  • SecurityInformationClass     Can be combination of:
  • OWNER_SECURITY_INFORMATION
  • GROUP_SECURITY_INFORMATION
  • DACL_SECURITY_INFORMATION
  • SACL_SECURITY_INFORMATION
  • DescriptorBuffer     Result of call - pointer to SECURITY_DESCRIPTOR structure.
  • DescriptorBufferLength     Size of buffer, in bytes.
  • RequiredLength     Pointer to value receiving required length of buffer.
