NtSetSecurityObject(



  IN HANDLE               ObjectHandle,
  IN SECURITY_INFORMATION SecurityInformationClass,
  IN PSECURITY_DESCRIPTOR DescriptorBuffer );



Function NtSetSecurityDescriptor writes object's Security Descriptor.


  • ObjectHandle     HANDLE to object of any type. Must be opened with WRITE_DAC or WRITE_OWNER access dependly to SecurityInformationClass parameter.
  • SecurityInformationClass     See NtQuerySecurityObject for possible values.
  • DescriptorBuffer     Pointer to user's allocated SECURITY_DESCRIPTOR to set.
