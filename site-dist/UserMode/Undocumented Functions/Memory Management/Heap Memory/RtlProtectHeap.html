RtlProtectHeap
RtlProtectHeap

NTSYSAPI 
PVOID
NTAPI


RtlProtectHeap(



  IN PVOID                HeapHandle,
  IN BOOLEAN              Protect );


HeapHandle
Address of heap.
Protect
If set, memory is protected to PAGE_READONLY. If zero, protect with PAGE_READWRITE.
Return value
Result is address of protected or unprotected heap.
