NTAPI


NtSetEaFile(



  IN HANDLE               FileHandle,
  OUT PIO_STATUS_BLOCK    IoStatusBlock,
  IN PVOID                EaBuffer,
  IN ULONG                EaBufferSize );



See NtQueryEaFile for information about EA.
  • FileHandle     HANDLE to File Object opened with FILE_SET_EA access.
  • IoStatusBlock     IO result of call.
  • EaBuffer     User's allocated input buffer containing one or more FILE_FULL_EA_INFORMATION structures.
  • EaBufferSize     Size of EaBuffer, in bytes.
