NtQuerySystemInformation
NtQuerySystemInformation

NTSYSAPI 
NTSTATUS
NTAPI


NtQuerySystemInformation(



  IN SYSTEM_INFORMATION_CLASS SystemInformationClass,
  OUT PVOID               SystemInformation,
  IN ULONG                SystemInformationLength,
  OUT PULONG              ReturnLength OPTIONAL );



NtQuerySystemInformation is used to check some system informations avaiable only in KernelMode (above 0x80000000). All avaiable (or all known) information classes are described in SYSTEM_INFORMATION_CLASS.


  • SystemInformationClass     Information class (see SYSTEM_INFORMATION_CLASS).
  • SystemInformation     User-allocated buffer for results. Sometimes this parameter can be NULL (OPTIONAL), if you check required buffer size (see below).
  • SystemInformationLength     Length of SystemInformation buffer (in bytes).
  • ReturnLength     Required length of SystemInformation buffer.
