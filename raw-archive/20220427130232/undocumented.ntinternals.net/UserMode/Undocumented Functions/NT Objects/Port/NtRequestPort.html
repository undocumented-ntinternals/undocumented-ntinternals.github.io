NTAPI


NtRequestPort(



  IN HANDLE               PortHandle,
  IN PLPC_MESSAGE         Request );



NtRequestPort sends request message to other side of LPC connection.
  • PortHandle     HANDLE to Port Object.
  • Request     Pointer to LPC_MESSAGE struct contains request data.
