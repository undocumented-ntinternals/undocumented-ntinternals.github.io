NtQueryVolumeInformationFile

NTSYSAPI

NTAPI


NtQueryVolumeInformationFile(



  IN HANDLE               FileHandle,
  OUT PIO_STATUS_BLOCK    IoStatusBlock,
  OUT PVOID               FileSystemInformation,
  IN ULONG                Length,
  IN FS_INFORMATION_CLASS FileSystemInformationClass );



  • FileHandle     HANDLE to File Object.
  • IoStatusBlock     IO result of call.
  • FileSystemInformation     Caller's allocated buffer for output data.
  • Length     Length of FileSystemInformation buffer, in bytes.
  • FileSystemInformationClass     Information class descripted in FS_INFORMATION_CLASS topic.


NtQueryVolumeInformationFile gives information about volume (device) containing file specified as FileHandle parameter.
