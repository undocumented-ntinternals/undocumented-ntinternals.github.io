NtSetIoCompletion(



  IN HANDLE               IoCompletionHandle,
  IN ULONG                CompletionKey,
  OUT PIO_STATUS_BLOCK    IoStatusBlock,
  IN NTSTATUS             CompletionStatus,
  IN ULONG                NumberOfBytesTransfered );



Function NtSetIoCompletion increments pending IO counter in IO Completion Object. It can be used to manual finish IO operation.


  • IoCompletionHandle     HANDLE to IO Completion Object opened with IO_COMPLETION_MODIFY_STATE access.
  • CompletionKey     User's defined key received by NtRemoveIoCompletion function.
  • IoStatusBlock     IO result of call.
  • CompletionStatus     IO operation status.
  • NumberOfBytesTransfered     Number of bytes transfered in manually finished IO operation.
